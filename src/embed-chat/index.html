Integration

<!-- Ajouter dans votre HTML -->
<div
  id="rooom-chat-root"
  data-studio-id="votre-studio-id"
  data-studio-name="Mon Studio"
  data-theme="light"
  data-accent-color="#6366f1"
  data-ai-name="YODA"
  data-position="bottom-right"
></div>
<script type="module" src="/embed-chat/main.js"></script>

Instructions

Cliquez sur la bulle en bas a droite pour ouvrir le chat.

Testez les differentes fonctionnalites : reservation, tarifs, escalade vers un humain.

Le widget s'adapte automatiquement au theme clair/sombre et a la couleur d'accent de votre site.
